Programming Works

Portfolio Website

使用ツール：Visual Studio Code
制作年 2024年 2025年 2026年

このポートフォリオサイト。
レスポンシブデザインを取り入れていて、PCでもスマートフォンでも快適に作品を見られるようにしている。
サイトの入り口になる「Home」ではざっくりと自分についての説明と、ジャンル問わず作品を羅列している。 Worksの画像をクリックすると該当ページにとぶこともできる。
作品のジャンルを「3DCG」「Animation」「Programming」「Others」と分けてそれぞれのページを作成し、上部メニューバー（またはハンバーガーメニュー）から遷移できるようにしている。
視覚的なストレスがないようなサイトを制作することを心がけ、作品ごとにブロックで囲って見やすくしている。作品の画像、映像は「3DCG」「Others」のページでは横2列、「Animation」「Programming」では1列で揃えている。 またどのページでも横幅が狭いとき縦一列で整列されるようにし、できるだけ作品が画面に大きく表示されるよう工夫している。
青字のリンクは新規タブで開くようにして元ページに戻る手間を省くことができるようにしている。 一方でジャンル間の移動は、新規タブで開くとかえって面倒になると考えたためそのまま遷移するようにしている。

伊咲薙 関連webサイト

使用ツール：Visual Studio Code
制作年 2025年

伊咲薙を語るストーリー、世界観の補助のためwebサイトを制作、公開し、伊咲薙がまるで現実に存在するものであるという表現をしている。
一昔前のサイトを思わせるために情報量は少なく、簡素なデザインにしたり、低画質な画像を使用したりしている。

サイト１「八籠神社 公式」:https://yatsukago-jinja.neocities.org
伊咲薙が撮影された神社である「八籠神社」の公式サイトという設定。

サイト2「朧猫の密蔵資料室」:https://oboro-secret-archives.neocities.org
伊咲薙を含む都市伝説をまとめている「朧猫」という人物の個人サイトという設定。

伊咲薙が実在する怪物であるという表現のために両サイトともにインターネットで「伊咲薙」や「八籠神社」といったワードで検索した際に引っかかるようにするためgoogleのindexに登録されるよう巡回申請をしている。 しかし思うように検索に反応せず、難しいと感じた。一方で、インターネット検索の仕組みについて理解を深めることができて良かったと思う。

※microsoft bingでは検索に出てきやすいです。（当方調べ）

Processing 作品集

使用ツール:Processing
制作年 2024年

Processingとは: Javaをベースにした、電子アートやビジュアルデザインの作成に特化した、プログラミング言語。

- 島のようなもの生成乱数 -
点の描写がランダムに動き回ることで、島の地図のようなものができるツール。ただプログラムを起動して待っていれば生成されるので操作は特にいらない。
ランダムに点が動き回る性質上、全く同じものができることは決してない。出来上がったものを「ここの湾、いい形してるなー」など言って楽しむこともできる。

作品: 島のようなもの生成乱数
別ウィンドウで開きます(PC推奨)

- ドット絵ツール -
マウスでカーソルを合わせてドラッグで黒色の描写ができて、何かキーを押すと白色の描写（消しゴム）ができるツール。20マス×20マスのドット絵を描くことができる。

作品: ドット絵ツール
別ウィンドウで開きます(PC推奨)

- 鬼ごっこゲーム -
赤鬼と青鬼から逃げるゲーム。勝利条件は特にない、ただ逃げるだけで面白みはないゲーム。赤鬼と青鬼の挙動がそれぞれ違うので注意。
WASDで移動。PC推奨。

作品: 鬼ごっこゲーム
別ウィンドウで開きます(PC推奨)

- 縦型STG -
敵機に攻め入られないように10機を撃墜することが目的。クリア条件は10機撃破。ゲームオーバーは被弾もしくは画面下部に敵機が侵入すること。
WASDで移動。space keyで自機弾発射（連射はできないので注意）。PC推奨。

作品: 縦型STG
別ウィンドウで開きます(PC推奨)
